# Networking Basics

## Introduction

Networking fundamentals form the backbone of distributed systems. Key concepts include OSI layers, TCP/UDP, HTTP, DNS, and how data moves between services.

## Key Concepts

| Concept | Definition |
|---------|------------|
| **TCP** | Connection-oriented, reliable, ordered delivery |
| **UDP** | Connectionless, no guarantee of delivery; low overhead |
| **HTTP** | Application protocol on TCP; request-response |
| **DNS** | Resolves domain names to IP addresses |
