# Read-Through Cache

## Introduction

**Read-through** cache sits in front of the database and loads on miss transparently. The application only talks to the cache.

## Definition

| Concept | Definition |
|---------|------------|
| **Read-Through** | Cache proxies reads; transparent to app; loads from DB on miss |
